
## Objective
This Java project contains multiple mini-applications demonstrating GUI programming and basic logic using Java Swing and console input.

## Project Overview
The project includes the following main Java classes:

- **ATMGUI.java**  
  A graphical user interface simulating ATM functionalities such as balance checking, withdrawal, and deposit.

- **DigitalLibraryManagement.java**  
  Manages digital library operations including adding, searching, and deleting books.

- **NumberGuessingGameGUI.java**  
  A GUI-based number guessing game providing hints until the correct number is guessed.

- **OnlineExamGUI.java**  
  A simple online exam application with multiple-choice questions and score calculation.
